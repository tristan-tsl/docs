# 背景介绍

自动化运维平台项目是为了提高运维效率、减少重复性的工作量从而达到提供总体的运维的水平、为公司节约人员、为上层业务提供更加快捷稳定的底层应用

后续会加入: 监控分析告警等功能, 从而为上层业务保驾护航
